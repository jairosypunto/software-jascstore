# README - Static files for Auths app

Esta carpeta está destinada a contener los archivos estáticos (CSS, JS, imágenes) 
relacionados con la aplicación `auths`.

Actualmente no hay archivos necesarios, pero este `readme.txt` asegura que la carpeta 
exista en el repositorio y sea reconocida por Django y Railway durante el despliegue.

Cuando se requiera, agrega aquí tus estilos o recursos específicos para el módulo de autenticación.
